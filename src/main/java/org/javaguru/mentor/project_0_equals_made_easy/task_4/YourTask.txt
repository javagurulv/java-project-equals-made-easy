Обратите внимание на параметр метода equals():

    public final boolean equals(Object obj) {
        return (this == obj);
    }

Object obj - это означает, что сюда теоретически
и практически можно передать ссылку на любой объект.

А это значит, что ваш объект можно попробовать
сравнить с любым другим объектом в Java программе.
