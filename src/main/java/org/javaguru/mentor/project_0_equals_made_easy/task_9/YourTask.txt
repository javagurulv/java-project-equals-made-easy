Если в методе equals() сравниваются две книги,
(проверка instanceof сработала), то далее
следует сравнить эти объекты по их свойствам.

Если все свойства одной книги совпадут со
свойствами другой книги тогда будем считать
такие объекты эквивалентными!

То есть два разных объекта книга с одинаковыми
значениями свойств будут считаться эквивалентными!

В классе Book дана полная реализация метода equals().

Загляните в тесты, там добавлены новые тесты на
проверку эквивалентности для каждого свойства!

Не забудьте повторить эти тесты для класса Car.
Практика это самое лучшее средство запоминания
новых идей и понятий.
