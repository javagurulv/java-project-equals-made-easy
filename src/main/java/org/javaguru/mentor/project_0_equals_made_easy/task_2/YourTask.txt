Пару слов про наследование в Java.

Наследование это один из двух способов
расширения функциональности программ.

Через наследование задается общее поведение
у иерархии классов.

В Java используют одиночное наследование, то есть
у каждого класса в Java есть только один родительский
класс. Исключением является класс Object, который
стоит во главе иерархии наследования в Java.

В классе Object как раз и определен метод equals()
о котором мы с вами и будем говорить в этом проекте.

Давайте взглянем на реализацию метода equals() в классе Object:

public boolean equals(Object obj) {
    return (this == obj);
}

Метод публичный, это значит что все наследники
унаследуют его при наследовании от класса Object.
Это значит, что этот метод есть у каждого класса
в Java!

Метод equals() предназначен для сравнения двух объектов
и определения являются ли они эквивалентными.

Его дефолтная реализация в классе Object сравнивает ссылки.

Если ссылки равны, то есть ссылаются на один и тот
же объект в памяти, то метод equals() вернет true,
иначе вернет false.
