Это простой учебный проект
по изучению эквивалентности объектов.

Изучите начальный код проекта.

В начальном коде представлены два класса:
- Book
- Car
а так же два тестовых класса
- BookTest
- CarTest

Мы будем рассказывать про метод equals()
используя класс Book и тестировать его
поведение используя тесты в классе BookTest.

Для закрепления материала повторяйте наши
действия с классом Car и его тестовым классом.
